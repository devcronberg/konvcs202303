    Konstanter

    Om modulet

    I dette modul vil du lære om de forskellige typer af konstaner i C#.

    Almindelige konstanter

    Faste værdier (kan ikke tilrettes efter tildeling)

    • variabler tildeles en værdi ved oprettelse
    • variabler kan ikke tildeles ny værdi
    • brug af const
    const int antalMånederPrÅr = 12;
const string applikationsNavn = "Min applikation";
Console.WriteLine($"antalMånederPrÅr = {antalMånederPrÅr}");
Console.WriteLine($"applikationsNavn = {applikationsNavn}");

// brug af indbyggede konstanter
// dette er nemmere (og mere sikkert)
double omkreds1 = System.Math.PI * 25;
Console.WriteLine($"omkreds1 = {omkreds1}");
// end dette
double omkreds2 = 3.141592653589792 * 25;
Console.WriteLine($"omkreds2 = {omkreds2}");

    Relaterede konstanter (enum)

    enum kan bruges til at gruppere relaterede konstanter i en såkaldt enumeration for at forbedre koden

    Vil gerne væk fra dette:

    string person1Navn = "Mathias";
int person1Alder = 12;
int person1Køn = 1; // antag 1 = mand

string person2Navn = "Lene";
int person2Alder = 53;
int person2Køn = 0; // antag 0 = kvinde

    og i stedet benytte en enumeration (bemærk - placeres typisk på namespace-niveau):

    public enum Køn
{
    Kvinde, // = 0
    Mand    // = 1
}

    kan bruges som

    string person1Navn = "Mathias";
int person1Alder = 12;
MitNamespace.Køn person1Køn = Køn.Mand;

string person2Navn = "Lene";
int person2Alder = 53;
MitNamespace.Køn person2Køn = Køn.Kvinde;

    Typekonvertering

    // Fra enum til nummer
Køn køn = Køn.Kvinde;
int kode = Convert.ToInt32(køn);

// Fra nummer til enum
kode = 1;
Køn køn2 = (Køn)kode;

    Andet om enums

    Værdier og type kan evt angives

    namespace MitNamespace
{
    public enum KortFarve : byte
    {
        Hjerter = 1,
        Ruder = 2,
        Spar = 4,
        Klør = 8
    }
}

    En del enums i frameworket

    System.DayOfWeek dag = DayOfWeek.Saturday;                // ugedage
System.ConsoleColor farve = ConsoleColor.Cyan;            // farve til konsol
System.IO.DriveType t = System.IO.DriveType.Network;      // drev type

    Enum-typen

    Brug eventuelt Enum-typen til at gennemløbe værdier eller navne

    foreach (var item in Enum.GetNames(typeof(DayOfWeek)))
{
    // string
    Console.WriteLine(item);
}

foreach (var item in Enum.GetValues(typeof(DayOfWeek)))
{
    // enum
    Console.WriteLine((int)item);
}

    Enum-typen indeholder også en Parse til typekonvertering:

    DayOfWeek d1 = (DayOfWeek)Enum.Parse(typeof(DayOfWeek), "Monday");
DayOfWeek d2 = (DayOfWeek)Enum.Parse(typeof(DayOfWeek), "monday", true);
