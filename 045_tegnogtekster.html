    Tegn og tekster

    Om modulet

    I dette modul lærer du om om tegn og tekster (strenge).

    Det er vigtigt at du får skrevet så meget kode som muligt - både ved at “lege” med eksemplerne fra sektionerne (kopier, kør og vær nysgerrig) samt ved at løse opgaverne.

    Enkelt tegn (char)

    .NET C# Forklaring
    System.Char char UniCode tegn (16 bit)


    Enkelt tegn

    • System.Char
    • char
    • omkranset af ‘ (ikke “)


    char a;
a = 'a';
Console.WriteLine($"a = {a}");

// a = 'ss';  = fejl
// a = "a";   = fejl


// ASCII værdi
int nr = Convert.ToInt32(a);
Console.WriteLine(nr);

// eller
nr = (int)'A';
Console.WriteLine(nr);


// den anden vej
char b = Convert.ToChar(65);
Console.WriteLine(b);

/*
 ---------- Output: ----------

a = a
97
65
A

*/

    Tekster (string)

    .NET C# Forklaring
    System.String string UniCode tekst


    • Tekst (2 gb)
    • omkranset af ” (og ikke ‘)
    string a;
a = "a";
a = "Kort sætning";
a = "Noget længere sætning - i øvrigt med danske tegn";

// erklæring og initialisering
string b = "test";

    Operatorer til manipulering

    Brug

    • brug ligmed til tildeling
    • brug ligmed ligmed (==) til sammenligning
    • alternativt String.Compare() - der returerer -1, 0 eller 1
    • brug plus til at sammenlægge
    • men pas på i løkker!!
    // Tildeling
string a;
a = "a";
Console.WriteLine($"a = {a}");

// Sammenligning
bool test;
test = a == "a";
Console.WriteLine($"test {test}");

Console.WriteLine($"ny test {a == "b"}");

string b = "b", c = "c";
int res = string.Compare(b, c);
Console.WriteLine($"Sammenlign b med c = {res}");
res = string.Compare(c, b);
Console.WriteLine($"Sammenlign c med b = {res}");
res = string.Compare(c, c);
Console.WriteLine($"Sammenlign c med c = {res}");


// Sammenlægning
a = a + " b";
Console.WriteLine($"a = {a}");
a += " c";
Console.WriteLine($"a = {a}");

    Masse metoder til streng manipulering

    string a;
a = "  Dette er en længere sætning  ";
Console.WriteLine($"Length = {a.Length}");
Console.WriteLine($"Contains = {a.Contains("sætning")}");
Console.WriteLine($"EndWith = {a.EndsWith("ning  ")}");
Console.WriteLine($"StartWith = {a.StartsWith("Test")}");
Console.WriteLine($"ToUpper = {a.ToUpper()}");
Console.WriteLine($"ToLower = {a.ToLower()}");
Console.WriteLine($"Trim = {a.Trim()}");
Console.WriteLine($"SubString = {a.Substring(2, 2)}");  // fra pos 2 og 2 frem

/*
 ---------- Output: ----------

Length = 31
Contains = True
EndWith = True
StartWith = False
ToUpper =   DETTE ER EN LÆNGERE SÆTNING  
ToLower =   dette er en længere sætning  
Trim = Dette er en længere sætning
SubString = De

*/

    Alle metoder returnerer en ny streng!!

    string a = "abc";
a.ToUpper();            // resultat benytte ikke
Console.WriteLine(a);   // abc
a = a.ToUpper();
Console.WriteLine(a);   // ABC

    Specialtegn

    foranstillet \

    Tegn Forklaring
    \b Backspace
    \f Formfeed
    \n New line
    \r Carriage return
    \t Horizontal tab
    \‘ Enkelt pling
    \“ Dobbelt pling
    \ Backslash
    \x hhhh Unicode karakter 
    string a;
// Tab
a = "\t1\t2";
Console.WriteLine($"a = {a}");

// Backslash
a = "c:\\temp\\fil.txt";
Console.WriteLine($"a = {a}");

// Ny linje
a = "\r\nLinje1\r\nLinje2\r\nLinje3\r\n";
Console.WriteLine($"a = {a}");

// Brug af @ for at få C# til at ignorere escape karaktere
a = @"c:\temp\fil.txt";
Console.WriteLine($"a = {a}");

/*
 ---------- Output: ----------

a =     1   2
a = c:\temp\fil.txt
a = 
Linje1
Linje2
Linje3

a = c:\temp\fil.txt

*/

    Interpolerede strenge (template)

    Brug \$ for at skabe en template:

    string navn = "Mathias";
string res = $"Hej - mit navn er {navn}";
Console.WriteLine(res);

int pris = 100;
DateTime dato = new DateTime(2018, 1, 1);
string res2 = $"Prisen er {pris} pr {dato}";
Console.WriteLine(res2);

string res3 = $"Prisen er {pris:n2} pr {dato:dd-MM-yyyy}";
Console.WriteLine(res3);

string res4 = $"Prisen er {pris.ToString("N2"), 10} pr {dato.ToString("dd-mm"),-10} og kan ikke forhandles.";
Console.WriteLine(res4);


/*
 ---------- Output: ----------

Hej - mit navn er Mathias
Prisen er 100 pr 01-01-2018 00:00:00
Prisen er 100,00 pr 01-01-2018
Prisen er     100,00 pr 01-00      og kan ikke forhandles.

*/

    Opgave 45-1

    Simpel opgave i brug af strenge (kode til søgning: EFB)

    Se opgave

    Strenge er en reference type

    Strenge er egentlig en refencetype så null kan forekomme. Derfor bør man checke om en streng er null inden brug:

    string a = null;
Console.WriteLine(a.ToUpper());     // fejl
if (a != null)
{
    Console.WriteLine(a.ToUpper()); // ok
}

    Andre operatorer

    Der findes et par andre operatorer man kan benytte for at forkorte koden:

    Operator Beskrivelse
    ?. Tester om string er null
    ?? Tester om ref er null - ellers returneres værdi 
    string a = null;
Console.WriteLine(a?.ToUpper());

    string a = null;
string b = a ?? "";
Console.WriteLine(b.ToUpper());

    Strenge er immutable

    PAS PÅ med manipulering af strenge i løkker

    Strenge er immutable

    • Immutabel (fast værdi / read only / kan ikke rettes)
    • Alle operationer på strenge returnerer en kopi
    • Performance kan være et problem

    Følgende kode tager eksempelvis over 1 minut at afvikle på en standard PC:

    // Stopur
System.Diagnostics.Stopwatch s = new System.Diagnostics.Stopwatch();
s.Start();
string a = "";
for (int i = 0; i < 500_000; i++)
{
    a += "*";
}
s.Stop();
Console.WriteLine($"ms = {s.ElapsedMilliseconds}");

    Brug System.Text.StringBuilder i stedet for (tager få millisekunder at afvikle):

    // Stopur
System.Diagnostics.Stopwatch s = new System.Diagnostics.Stopwatch();
System.Text.StringBuilder sb = new System.Text.StringBuilder();
s.Start();
for (int i = 0; i < 500_000; i++)
{
    sb.Append("*");
}
s.Stop();
Console.WriteLine($"ms = {s.ElapsedMilliseconds}");

    System.Text.StringBuilder

    Her er et par korte eksempler på brug af System.Text.StringBuilder:

    System.Text.StringBuilder sb = new System.Text.StringBuilder();
sb.AppendLine("Linje 1");
sb.AppendLine("Linje 2");
sb.Insert(3, "**");
sb.Replace("**", "!!");
sb.Append("@@");
Console.WriteLine($"Længde: {sb.Length}");
Console.WriteLine($"Indhold:\r\n{sb.ToString()}");

/*
 ---------- Output: ----------

Længde: 22
Indhold:
Lin!!je 1
Linje 2
@@

*/

    NuGet-pakker

    Der findes en del forskellige pakker relateret til brug af strenge. Se eksempelvis Humanizr/Humanizer (pakken hedder Humanizer og den lokaliserede del hedder Humanizer.Core.Da) for yderligere streng funktionalitet, samt Handlebars-Net/Handlebars.Net (pakken hedder Handlebars.Net) for brug af skabeloner.

    Eksempler på brug af Humanizer (bemærk - dansk er ikke fuldt oversat):

    using System;
using System.Globalization;
using Humanizer;

namespace NuGetTest
{
    internal class Program
    {
        static void Main(string[] args)
        {

            // Sæt kultur (ikke fuld oversat)
            System.Threading.Thread.
                CurrentThread.CurrentUICulture =
                new CultureInfo("da-DK");

            string a = DateTime.Now.AddDays(-10).Humanize();
            Console.WriteLine(a);

            string c = "Lang tekst som skal forkortes".Truncate(15, " ...");
            Console.WriteLine(c);

            TimeSpan d = TimeSpan.FromMinutes(60);
            Console.WriteLine(d.Humanize());

            Console.WriteLine((360.0).ToHeading(HeadingStyle.Full));
            Console.WriteLine((130.0).ToHeading(HeadingStyle.Full));

        }
    }
}
