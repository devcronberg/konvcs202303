    Samlinger

    Om modulet

    Her skal du lære om forskellige samlinger der kan bruges til at opbevare data. Der findes mange forskellige men du kan nøjes med at fokusere på de mest benyttede:

    • System.Collections.Generic.List<>
    • System.Collections.Generic.Queue<>
    • System.Collections.Generic.Stack<>
    • System.Collections.Generic.Dictionary<>

    De resterende klasser (under System.Collections og System.Collections.Generic) kan være meget effektive, men de er ikke brugt så meget i grundlæggende C#.

    Overordnet om samlinger

    Der findes mange forskellige strukturer der kan benyttes til opbevare data i C#. System.Array (standard array) er en type, men C# tilbyder en masse andre og langt mere effektive strukturer. I de første versioner af .NET var klasser under System.Collection de eneste tilgængelige, men i de senere versioner er samlinger (collections) under System.Collections.Generic dukket op. De er langt mere effektive end på arrays og “de gamle” collection-klasser, og du bør derfor fokusere på disse.

    Typesvage samlinger

    • Findes en del Collection-klasser under System.Collection - bla

    • ArrayList

    • HashTable
    • SortedList
    • Stack

    • Queue

    • Benyttes ikke så meget mere fordi de er typesvage - brug i stedet System.Collections.Generic-klasser

    System.Collections.ArrayList lst =
    new System.Collections.ArrayList();
lst.Add("mathias");
lst.Add("mikkel");
lst.Add(12);
lst.Add(true);
foreach (var item in lst)
{
    Console.WriteLine(item);
}

/*
 ---------- Output: ----------

mathias
mikkel
12
True

*/

    Typestærke samlinger

    Anderledes er det med de typestærke collections fra System.Collections.Generics

    • Benyt <> for at angive typen
    • Mange forskellige muligheder
    • List<>
    • Stack<>
    • Queue<>
    • Hash<>
    • Dictionary<>
    • Hvis man har “using System.Linq;” tilføjes en masse ekstra metoder - mere om det senere. For at simplificere forståelsen vil jeg anbefale, at du fjerner “using System.Linq;” indtil du kender de enkelte collection-klasser.

    List<>

    En List<> kan sammenlignes med et dynamisk array, og indeholder en masse metoder. Collection-klassen er langt mere effektiv end arrays, og du vil sikkert benytte denne klasse til at opbevare data i 80-90 procent af tilfældene.

    System.Collections.Generic.List<string> lst = new System.Collections.Generic.List<string>();
lst.Add("a"); // kan kun håndtere en streng
// lst.Add(1);  fejl
lst.Add("b");
string c = "c";
lst.Add(c);

lst.Insert(1, "*");
lst.InsertRange(2, new string[] { "q", "ww", "ee", "rr" });
lst.Remove("*");
lst.RemoveAt(2);
Console.WriteLine($"Findes q: {lst.Contains("q")}");

Console.WriteLine("Vis alle:");
foreach (var item in lst)   // item = string
{
    Console.WriteLine(item);
}

lst.Sort();
Console.WriteLine();
Console.WriteLine("Vis alle efter sort:");
foreach (var item in lst)   // item = string
{
    Console.WriteLine(item);
}

// avanceret (delegates/lambda - se senere)
var res = lst.FindAll(s => s.Length == 1);
Console.WriteLine($"Hvor mange elementer har længden 1: {res.Count}");


/*
 ---------- Output: ----------

Findes q: True
Vis alle:
a
q
ee
rr
b
c

Vis alle efter sort:
a
b
c
ee
q
rr
Hvor mange elementer har længden 1: 4

*/

    Dictionary<,>

    • Nøgle og værdi struktur
    • Der skal både oprettes en type af nøglen og en type af værdien
    // nøgle = int, værdi = string
System.Collections.Generic.Dictionary<string, int> lst = new System.Collections.Generic.Dictionary<string, int>();
lst.Add("a", 1324);
lst.Add("b", 23);
lst.Add("c", 113);
foreach (var item in lst)
{
    Console.WriteLine($"{item}, {item.Key}, {item.Value}");
}
int v = lst["b"];
Console.WriteLine($"v = {v}");

/*
 ---------- Output: ----------

[a, 1324], a, 1324
[b, 23], b, 23
[c, 113], c, 113
v = 23

*/

    IEnumerable

    Giver mulighed for, at en klasse kan itereres, og er implementeret af de fleste collection-klasser og arrays. Består af metoden GetEnumberator, som returnerer IEnumerator

    • Current
    • MoveNext() (returnerer false hvis der ikke er flere elementer)
    • Reset()
    static void Main(string[] args)
{
    IEnumerable<int> tal = FindTal1();
    IEnumerator<int> enumerator = tal.GetEnumerator();
    while (enumerator.MoveNext())
        Console.WriteLine(enumerator.Current);
}

static IEnumerable<int> FindTal1()
{
    List<int> lst = new List<int>();
    int i = 0;
    do
    {
        i++;
        lst.Add(i);
        Console.WriteLine($"Fundet {i} i FindTal1");
        if (i == 3)
            break;
    } while (true);
    return lst;
}

    Men man kan også bruge interfacet selv.

    class Person
{
    public string Navn { get; set; }
    public int Alder { get; set; }
}

class Personer : IEnumerable<Person>
{
    private List<Person> _personer = new List<Person>();

    public IEnumerator<Person> GetEnumerator()
    {
        return ((IEnumerable<Person>)_personer).GetEnumerator();
    }

    public void TilføjPerson(Person p)
    {
        _personer.Add(p);
    }

    IEnumerator IEnumerable.GetEnumerator()
    {
        return ((IEnumerable<Person>)_personer).GetEnumerator();
    }
}

    Personer lst = new Personer();
lst.TilføjPerson(new Person { Navn = "a", Alder = 1 });
lst.TilføjPerson(new Person { Navn = "b", Alder = 2 });
lst.TilføjPerson(new Person { Navn = "c", Alder = 3 });
foreach (var p in lst)
    Console.WriteLine(p.Navn);

    Brug af yield

    yield-kodeordet kan benyttes i metoder/egenskaber til at skabe et resultat (IEnumerable) der kan itereres. Der kan yield return fra en metode bruges til at returnere et element af gangen. Det kan benyttes til at returnere en strøm af data i stedet for en komplet datastruktur. Samtidigt skaber yield en statemachine som kan benyttes i forskellige alogritmer.

    Se følgende - og læg mærke til forskellen i de to måder at danne tal

    static void Main(string[] args)
{
    {
        IEnumerable<int> tal = FindTal1();
        IEnumerator<int> enumerator = tal.GetEnumerator();
        while (enumerator.MoveNext())
            Console.WriteLine(enumerator.Current);
    }
    Console.WriteLine();
    {
        IEnumerable<int> tal = FindTal2();
        IEnumerator<int> enumerator = tal.GetEnumerator();
        while (enumerator.MoveNext())
            Console.WriteLine(enumerator.Current);
    }
    Console.WriteLine();
    // exit
    {
       IEnumerable<int> tal = FindTal2();
       IEnumerator<int> enumerator = tal.GetEnumerator();
       int i = 0;
       while (enumerator.MoveNext())
       {
        Console.WriteLine(enumerator.Current);
        if (i++ == 1)
            break;
       }
    }    

    // kan også implementeres med foreach
    Console.WriteLine();
    {
        foreach (var item in FindTal1())
            Console.WriteLine(item);
    }
    Console.WriteLine();
    {
        foreach (var item in FindTal2())
            Console.WriteLine(item);
    }

}

static IEnumerable<int> FindTal1()
{
    List<int> lst = new List<int>();
    int i = 0;
    do
    {
        i++;
        lst.Add(i);
        Console.WriteLine($"Fundet {i} i FindTal1");
        if (i == 3)
            break;
    } while (true);
    return lst;
}

static IEnumerable<int> FindTal2()
{
    int i = 0;
    do
    {
        i++;
        Console.WriteLine($"Fundet {i} i FindTal2");
        yield return i;
        if (i == 3)
            yield break;
    } while (true);
}

    Her er et andet eksempel på brug af yield til at finde data (fra db, fil, logik mv)

    class Person
{
    public string Navn { get; set; }
    public int Alder { get; set; }
}

class Personer : IEnumerable<Person>
{
    private List<Person> _personer = new List<Person>();

    public IEnumerator<Person> GetEnumerator()
    {
        return ((IEnumerable<Person>)_personer).GetEnumerator();
    }

    public void TilføjPerson(Person p)
    {
        _personer.Add(p);
    }

    public IEnumerable<Person> FindAllePersoner()
    {
        foreach (var p in _personer)
            yield return p;
    }

    public IEnumerable<Person> FindPersonerUnderAlder(int alder)
    {
        foreach (var p in _personer)
            if (p.Alder < alder)
                yield return p;
    }
    IEnumerator IEnumerable.GetEnumerator()
    {
        return ((IEnumerable<Person>)_personer).GetEnumerator();
    }
}

    Personer lst = new Personer();
lst.TilføjPerson(new Person { Navn = "a", Alder = 1 });
lst.TilføjPerson(new Person { Navn = "b", Alder = 2 });
lst.TilføjPerson(new Person { Navn = "c", Alder = 3 });

foreach (var p in lst)
    Console.WriteLine(p.Navn);

foreach (var p in lst.FindAllePersoner())
    Console.WriteLine(p.Navn);

foreach (var p in lst.FindPersonerUnderAlder(2))
    Console.WriteLine(p.Navn);
